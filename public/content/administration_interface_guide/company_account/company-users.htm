Company Users

To open this page, go to Company Account>Company Users page.

This page is where you can add and manage company users. Generally, you need to create and assign users to companies after a company and its business units have been created.

From this page you can:

Create company users

Edit company users

...

Graphic User Interface

The Company Users page consists of the following elements:

# Element Description
1 +Add User This button adds a new user. See Creating a New User for more details.
2 Show [number] entries Here you can select how many items you want to see in the list of navigation elements (10, 25, 50, 100).
3 User ID Numeric identifier of the merchant relation.
4 Company Name Name of the company the user belongs to.
5 User Name Name of the company user.
6 Roles Roles the user has been assigned to. See Company Roles to learn how to create a new company role.
7 Company Business Units Business units the company user is assigned to.
8 Status Current status of the user. See to learn more about statuses a company user can have.
9 Actions In this column you can Edit Disable or Delete a company user. See Managing Company Users for more details.
10 Search Search and filter company users. Start typing a merchant relation ID, user name or company name to filter the results and display them by ID, Company Name and User Name in the List of Company Users table.
11 Showing [number 1] to [number 2] of [number 3] entries Shows how many entries ([number 1] to [number 2]) are displayed in the table. [number 2] depends on the value set in Show [number] entries (2), [number 3] represents total number of entries in the company users table.
12

Previous [page number] Next

[page number] is the number of page you are currently on. Here you can switch to the previous page or to the next page.
