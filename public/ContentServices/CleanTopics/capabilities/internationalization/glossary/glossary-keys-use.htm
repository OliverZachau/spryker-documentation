One of the usages of the glossary keys in the Shopp Application is for rendering translated content. For allowing to render translated content, a dedicated extension for the twig template engine is available to be used. You can see bellow how you can add translated content in Yves <div class="form-group"><label class="col-sm-2 control-label">First Name</label> <div class="col-sm-10"> <input type="text" class="form-control" name="first_name" value="{{ form.first_name.value }}"> {#{{ form_row(form.first_name) }}#} </div> </div> The keywordtransmarks an operation exposed by the TwigTranslator extension; this means that the text of the label will either be the default value provided in the view or the corresponding translation for it.
