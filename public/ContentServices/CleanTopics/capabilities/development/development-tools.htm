The Spryker Commerce OS provides your developers with a range of coding tools that make sure your OS run smoothly. - Code sniffer: This will validate the code style - Architecture sniffer: Checks whether the code follows the Spryker architecture - Code generator: Lets you generate modules or parts of modules (application layers) - PHPStan: Static code analyzer for code introspection, helps to reduce bugs The OS is fully equipped with a production/ staging deployment solutions for different cloud platforms, like AWS.
