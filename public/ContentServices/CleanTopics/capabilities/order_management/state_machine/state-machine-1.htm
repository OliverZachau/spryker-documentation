TheStateMachinemodule provides generic implementation for state machines (SM). This module provides functionality for drawing the SM graph, triggering events, initializing a new state machine or for getting the state history for a processed item. If you are looking for information on the OMS State Machine see: OMS State Machine There is already example state machine implemented in demoshop. Look for stateMachineExample module, you can use as a base for implementing your custom SM.
