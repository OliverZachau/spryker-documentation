"Allow your sales agents to create orders manually. In the Administration Interface users can effortlessly create orders, enter the relevant checkout information and place the order on behalf the customer. The order entry form, out of the box, is optimized for processing orders via phone and can be customized as required by adding new fields from other modules. A sales agent can easily modify the prices in the manual order creation workflow, to accommodate customer specific prices or discounts." Zed User is able to create an order, fulfill with cart details and checkout information and place it. "- Order entry from is optimized for sales manager that creating order by contacting customer by phone. - Process represent one page checkout. - Ability to force prices for order items (base item price) - Order entry form is extendible with new fields from other modules "
