In order to reproduce some edge cases like declined payment or pending capture, Amazon provides two solutions. The first is special methods marked with red star on payment widget. It allows to reproduce different cases of "decline" payment workflow. But there are more edge cases like expired authorization or pending capture for which there is only one way to reproduce - pass simulation string as 'SellerNote' parameter of API request.
