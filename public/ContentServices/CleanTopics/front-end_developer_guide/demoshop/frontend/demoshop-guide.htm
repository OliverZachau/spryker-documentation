Bootstrap the project To install all frontend the external dependecies for Yves and Zed, run in the VM console: cd /path/to/project/root ./setup -i Yves To build assets for Yves, run in the console: cd /path/to/project/root npm run yves # build assets in development mode npm run yves:dev # build assets in development mode with watchers over the code npm run yves:prod # build assets in production mode Inside and outside the VM You can run these scripts inside and outside the VM. Run them outside the VM if you want to improve the build performance and the watchers speed. Zed To build assets for Zed, run in the console: cd /path/to/project/root npm run zed # build assets in development mode npm run zed:dev # build assets in development mode with watchers over the code npm run zed:prod # build assets in production mode Inside and outside the VM You can run these scripts inside and outside the VM. Run them outside the VM if you want to improve the build performance and the watchers speed. Custom scripts If you need to add or change the scripts, open the@project/package.jsonmanifest file and change the related section: "scripts": { "yves": "node ./assets/Yves/default/build", "yves:dev": "node ./assets/Yves/default/build --dev", "yves:prod": "node ./assets/Yves/default/build --prod", "zed": "node ./node_modules/@spryker/oryx-for-zed/build", "zed:dev": "node ./node_modules/@spryker/oryx-for-zed/build --dev", "zed:prod": "node ./node_modules/@spryker/oryx-for-zed/build --prod" }
