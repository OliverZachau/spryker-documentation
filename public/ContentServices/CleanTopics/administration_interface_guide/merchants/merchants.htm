To open this page, go toMerchants -> Merchants This page is where you can add and manage merchants. Merchants are actually sellers usually represented by a company. From this page you can: Create merchants Edit merchants Delete merchants Graphic User Interface TheMerchantspage consists of the following elements: # Element Description 1 +Add Merchant This button adds a new merchant. See Managing Merchants for more details. 2 Show [number] entries Here you can select how many items you want to see in the list of navigation elements (10, 25, 50, 100). 3 Search Search and filter merchants. Start typing a merchant ID or name to filter the results and display them by Merchant Id and Name in theList of Merchantslist. 4 Merchant Id Merchant identifier. 5 Name Name of the merchant. 6 Actions In this column you can eitherEditorDeletea merchant. See Managing Merchants for more details. 7 Showing [number 1] to [number 2] of [number 3] entries Shows how many entries ([number 1] to [number 2]) are displayed in the table. [number 2] depends on the value set in Show [number] entries (2), [number 3] represents total number of entries in the merchants table. 8 Previous [page number] Next [page number] is the number of page you are currently on. Here you can switch to the previous page or to the next page. Managing Merchants Tocreate a merchant, do the flowing: Click+Add Merchantin the top right corner. Enter Merchant name and hitSave . The created merchant will appear on theOverview of Merchantspage. Toedit a merchant, do the following: In theList of Merchantstable (Overview of Merchantspage), findActionscolumn and clickEditfor a merchant you want to edit. On theEdit Merchantpage, edit the merchant name and hitSave. Todelete a merchant, in theList of Merchantstable (Overview of Merchantspage), findActionscolumn and clickDeletefor a merchant you want to delete. The merchant will be deleted.
